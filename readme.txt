== Description ==
This Sidebar widget will monitor and display products from
Woot.com
Kidswoot.com
Winewoot.com
Shirtwoot.com
Sellout.com


Some of the features:
* Automatically refresh for Woot Off
* Select the woots you want to show
* Compatible with Wordpress (have not tested on BuddyPress or Mu)
* Will work with any caching system.

== Installation ==
1. Upload this plugin to the `/wp-content/plugins/` directory and unzip it, or
simply upload the zip file within your wordpress installation.

2. Activate the plugin through the 'Plugins' menu in WordPress

3. If you want to add other woots go to the "Settings -> Woot Watcher" and
select the ones to display.

4. Add the "Woot Watcher" with ajax widget to your sidebar.

5. Happy WOOTING!!
